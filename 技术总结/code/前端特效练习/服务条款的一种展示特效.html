孤独者

蒋勋在《孤独六讲》中讲到思维孤独。

生活需要思考，思考带来收获，正是因为前人锲而不舍的苦苦思想，才有了（什么样的）今天，当时的他们也是不被人理解的，作为一个思维孤独者，面临着多方面的压力，以至于至今伟大的思想者也没多少，在物欲横流的今天，我们怎样保持内心不被外界所影响，做一个高傲的思维孤独者。

得失之间

人生，困扰我们最多的，无非就是得失这两个字。

一切纷争、烦恼、迷茫的根源，也是这两个字。常言说，"得失之间，必有因果"。.只要我们懂得了得失的原因，自然就明白了取舍之道，然后，一切问题迎刃而解。

心若向阳

若隐若现的场景，曾几度时梦到你的身影，未知的面貌，未知的开始。

我在时光里莅位，期待，在每个轻风云淡是时光里，写出锦瑟生香的文字，依着阳光安暖，悄然长成一束的花，等春天的情话在时光里栖霞，你眼中的那寂寞的爱贴，便是我心若向阳的开始。
